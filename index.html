About Me

I started my professional journey as an electronics engineer. I wanted to build guitar amps and audio equipment but during my graduate studies, found out I enjoyed writing audio software more. I transitioned into software development and there found a love of designing user interfaces. Now, I'm building websites for my own projects and (eventually) people in my local community.

I've always been meticulous in whatever I do, whether it's learning how to make sauerkraut, developing photographic film, or designing a new software feature. I'm a big proponent of clean code and architecture and enjoy a good code review almost as much as writing the code itself. Just as important as the back end is making getting the user-facing design right. I love reading about graphic design, art, and photography and these influence my work in web design. Sustainability is another important value I hold, so I make sure to follow green software development practices on my own projects.

Take a look at the projects listed below if you'd like an idea of what I've worked on. You can find more information about the tools I used for each project on each page. I'm looking for new projects, so don't hesitate to reach out!
